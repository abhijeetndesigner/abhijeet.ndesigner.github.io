# Abhijeet Portfolio

This is Abhijeet's professional portfolio built with Next.js, React, TailwindCSS, and custom design tokens.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

This portfolio is configured for static export and automated deployment via GitHub Actions to GitHub Pages.

### Configuration
The base path is configured for the domain root: `https://abhijeetndesigner.github.io/`.
The deployment workflow is defined in `.github/workflows/deploy.yml`.
The repository is hosted at `https://github.com/abhijeetndesigner/abhijeetndesigner.github.io`.
